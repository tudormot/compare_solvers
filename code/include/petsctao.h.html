 69: /*E
 70:     TaoConvergedReason - reason a TAO method was said to have converged or diverged

 72:    Level: beginner

 74:    The two most common reasons for divergence are
 75: $   1) an incorrectly coded or computed gradient or Hessian
 76: $   2) failure or lack of convergence in the linear system (in this case we recommend
 77: $      testing with -pc_type lu to eliminate the linear solver as the cause of the problem).

 79:    Developer Notes: this must match petsc/finclude/petsctao.h

 81:        The string versions of these are in TAOConvergedReasons, if you change any value here you must
 82:      also adjust that array.

 84: .seealso: TAOSolve(), TaoGetConvergedReason(), KSPConvergedReason, SNESConvergedReason, TSConvergedReason
 85: E*/
 86: typedef enum {/* converged */
 87:   TAO_CONVERGED_GATOL         =  3, /* ||g(X)|| < gatol */
 88:   TAO_CONVERGED_GRTOL         =  4, /* ||g(X)|| / f(X)  < grtol */
 89:   TAO_CONVERGED_GTTOL         =  5, /* ||g(X)|| / ||g(X0)|| < gttol */
 90:   TAO_CONVERGED_STEPTOL       =  6, /* step size small */
 91:   TAO_CONVERGED_MINF          =  7, /* F < F_min */
 92:   TAO_CONVERGED_USER          =  8, /* User defined */
 93:   /* diverged */
 94:   TAO_DIVERGED_MAXITS         = -2,
 95:   TAO_DIVERGED_NAN            = -4,
 96:   TAO_DIVERGED_MAXFCN         = -5,
 97:   TAO_DIVERGED_LS_FAILURE     = -6,
 98:   TAO_DIVERGED_TR_REDUCTION   = -7,
 99:   TAO_DIVERGED_USER           = -8, /* User defined */
100:   /* keep going */
